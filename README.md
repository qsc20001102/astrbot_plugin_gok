# 王者数据查询工具

## 简介

主要是自用，查询战绩，在游戏中隐藏的战绩也可以查，如果营地的战绩也隐藏了就查不了。只有营地里面大号的ID才可查询。

插件功能绝大部分都是用AI写的，又进行了调试。

插件数据依赖外部接口提供，需要自行支持创建令牌才可使用。（免费调用）


## 功能特点

- **处理接口数据**：将接口返回的数据处理成，玩家能读懂的消息发送
- **HTML渲染图片**：利用astrbot里面的文转图服务，处理HTML页面渲染成图片

## 插件配置

**指令前缀**

开启后必须要在指令功能前面加上下面配置的前缀才能正常触发。关闭后直接只有指令功能即可。

**应天API 令牌**

前往https://api.t1qq.com 注册并创建令牌（免费）

**柠柚API 令牌**

前往https://api.nycnm.cn 注册并创建令牌（免费）

## 使用方式

如果开启了前缀，需要在所有指令前面加上设定的前缀。

在聊天中输入 **功能** 即可获取插件所有的功能使用说明。

在使用 **战绩** 和 **资料** 两个功能时后面可以直接输入营地ID进行查询，或者输入提前自定义好的角色来查询。

在使用 **上榜战力** 时，可以在英雄名称后面加一个大区参数 aqq awx iqq iwx 四个大区，不写默认aqq。

指令 **角色查看**、**角色添加**、**角色修改**、**角色删除**、**角色查询** 就是用来操作角色数据的，给王者营地ID起一个别名，方便自己记忆，也方便查询。
